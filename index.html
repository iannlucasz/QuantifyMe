🎮 Quantifiers Game

Neste jogo, você vai praticar os quantifiers (quantificadores) em inglês! Eles servem para indicar quantidade e aparecem antes de substantivos. Veja alguns exemplos:

  • much → usado com substantivos incontáveis
    Ex: I don’t have much money.
  • many → usado com substantivos contáveis no plural
    Ex: She has many friends.
  • a little → pequena quantidade (incontáveis)
    Ex: There’s a little juice left.
  • a few → pequena quantidade (contáveis)
    Ex: I have a few pencils.
  • some / any → quantidade indefinida (some para afirmativa, any para negativa/interrogativa)
  • too / enough → intensidade (demais / o suficiente)
