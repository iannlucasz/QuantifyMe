Quantifiers Game 🎯

Aprenda a usar quantificadores como much, many, some, a few e mais!

  • Much: usado com substantivos incontáveis
  • Many: usado com contáveis no plural
  • A few / A little: pequena quantidade (contáveis/incontáveis)
  • Some / Any: quantidades indefinidas
  • Too / Enough: intensidade (demais / suficiente)
